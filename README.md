# Gajanand Kumar — Portfolio

Premium, fully responsive personal portfolio for **Gajanand Kumar (Gajju)** — BCA student, developer, and tech enthusiast based in Muzaffarpur, Bihar, India.

## Tech stack

- **React 18** + **TypeScript**
- **Vite** build tool
- **Tailwind CSS** (dark luxury theme, glassmorphism)
- **Framer Motion** animations
- **PWA** support via `vite-plugin-pwa`

## Features

- Dark/light mode toggle
- Custom cursor (desktop)
- Animated particle background
- Scroll-reveal animations
- Project filtering
- Animated statistics
- Download Resume button
- SEO meta tags + Open Graph
- Installable PWA

## Getting started

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Customize

Most personal data lives in `src/data.ts`. Search the codebase for `PLACEHOLDER` and replace social links, email, resume, profile image, PWA icons and project links.
